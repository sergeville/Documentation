# Markdown Cheat Sheet

---

## Headings

```
# H1
## H2
### H3
#### H4
```

---

## Text Formatting

```
**bold**
*italic*
~~strikethrough~~
`inline code`
```

**bold** — *italic* — ~~strikethrough~~ — `inline code`

---

## Lists

```
- Item
- Item
  - Nested item

1. First
2. Second
3. Third
```

---

## Links & Images

```
[Link text](https://example.com)
![Alt text](image.png)
```

---

## Blockquote

```
> This is a quote
> spanning multiple lines
```

> This is a quote

---

## Code Block

````
```python
def hello():
    print("Hello, world!")
```
````

```python
def hello():
    print("Hello, world!")
```

## Table

```
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell     | Cell     | Cell     |
| Cell     | Cell     | Cell     |
```

| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell     | Cell     | Cell     |
| Cell     | Cell     | Cell     |

## Horizontal Rule

```
---
```

---

## Checkboxes

```
- [ ] Todo item
- [x] Done item
```

- [ ] Todo item
- [x] Done item

---

## Footnote

```
Here is a sentence.[^1]

[^1]: This is the footnote.
```

---

## Escape Special Characters

```
\*  \#  \_  \`  \[  \]
```

Prefix any special character with `\` to display it literally.
